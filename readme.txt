Wishi - Telegram Wishlist Bot

Опис
Wishi - це Telegram-бот, який дозволяє користувачам створювати власний вішліст, переглядати бажання інших користувачів та керувати своїм списком бажань.

Функціональність
- Додавання бажань у вішліст
- Перегляд власного вішліста
- Видалення бажань з вішліста
- Перегляд вішлісту інших користувачів за ID або нікнеймом
- Отримання власного ID
- Автоматична реєстрація користувачів при першому запуску

Використані технології
- Python 3
- Telegram Bot API
- MongoDB (pymongo) для зберігання даних

Встановлення та запуск
1. Клонування репозиторію
2. Встановлення залежностей
3. Запуск сервера MongoDB
4. Налаштування токена бота
Зареєструйте нового бота через [BotFather](https://t.me/BotFather) та отримайте API-токен. Впишіть його у змінну `API_TOKEN` у файлі `bot.py`.
5. Запуск бота


Команди бота
| Команда          | Опис                                      |
|-----------------|-----------------------------------------|
| `/start`        | Запуск бота та реєстрація користувача  |
| `/add`          | Додати бажання                         |
| `/view`         | Переглянути свої бажання               |
| `/delete`       | Видалити бажання з вішліста            |
| `/view_user`    | Переглянути вішліст іншого користувача |
| `/my_id`        | Отримати свій Telegram ID              |

Структура бази даних
**Колекція `users` (Користувачі):**
```json
{
  "user_id": 123456789,
  "username": "example_user",
  "first_name": "John",
  "last_name": "Doe"
}
```

**Колекція `wishlists` (Вішлісти):**
```json
{
  "user_id": 123456789,
  "item": "Новий ноутбук"
}
```
